Cohsa

A single-page html+coffee+sass+jsv framework to be used with Phonegap and Node Webkit

1. Install CJSV.

clone https://github.com/ph-everywhere/cjsv to any directory you like

2. Install CoffeeScript and Compass

3. Install Cohsa.

clone https://github.com/ph-everywhere/cohsa to any directory you like

4. Set up environment

Add these lines to your .bashrc or similar.

alias c_jw='ruby your/path/to/jsv/jsv.rb --input_dir application/views/cjsv/ --output_dir application/views/'

alias c_csw='coffee -wc -o js/ .'

alias cw='compass watch .'

alias cohsa='ruby your/path/to/cohsa/tools/tools.rb'

5. Create your project

Cohsa is independent of Phonegap and Node Webkit. If you are using phonegap, this creation should be on www/ directory.

cohsa create path/to/name_your_project

Open three terminals (Or a terminal that allows screen splashing, like Terminator) on the just created project folder, and run the first three previously set aliases. On this step you will probably get a lot of ruby dependency related problems. I'm counting on you to solve them!

Point you browser to index.html and you should see a nice welcome message.
